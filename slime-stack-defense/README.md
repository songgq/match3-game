# 史莱姆叠塔守卫

手机竖屏 HTML5 小游戏原型：物理堆叠俄罗斯方块 + 自动塔防。

## 玩法

- 从底部三张卡牌中拖出俄罗斯方块，松手后由物理引擎落下。
- 方块必须通过堆叠连接到底部承重平台，并保持稳定，史莱姆才会开火。
- 方块倾斜超过约 45° 会停止攻击；从平台两侧坠落后会被清除。
- 怪物沿“右上 → 左上 → 左下 → 右下”路线移动。
- 普通怪到达终点后伤害核心并消失，Boss 会停下持续攻击。
- 守住五波怪物并击败最终 Boss 即可获胜。

## 内容

- 长弓、爆弹、秘法、重炮、冰霜五类史莱姆职业。
- 普通、迅捷、铁甲、精英与 Boss 敌人。
- 能量循环、连击、粒子反馈、屏幕震动、程序化音乐与音效。
- Matter.js 2D 物理与 Canvas 矢量绘制，无图片素材依赖。
