# 消消乐 (Match-3 Game)

一个纯前端实现的经典消消乐游戏，使用 HTML5 Canvas + 原生 JavaScript 构建。

## 在线试玩

通过 GitHub Pages 托管，点击即可游玩。

## 功能

- **8×8 棋盘**，6 种宝石
- 点击相邻方块交换位置
- 3 个及以上相同宝石连成一线即可消除
- 连击加分机制
- 30 步限制，用完即结束
- 流畅动画效果
- 适配移动端和桌面端

## 本地运行

直接打开 `index.html` 即可，无需任何构建步骤。

```
npx serve .
```

## 部署到 GitHub Pages

1. 创建新仓库并推送代码
2. 在仓库 Settings → Pages → Source 选择 `main` 分支

## 技术栈

- HTML5 Canvas
- CSS3 (Flexbox, Gradient, Backdrop Filter)
- 原生 JavaScript (ES6+)
